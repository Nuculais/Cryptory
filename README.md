CRYPTORY
============

## REQUIREMENTS
- docker 18.x
- docker-compose 18.x
- yarn v1.6.x
- node v8.11.x

## TO RUN
```shell
$ yarn install
$ yarn run local
```
#### OR
```shell
$ docker-compose build
$ docker-compose uo
```

## WHAT IT IS
A way to keep track of your cryptocurrencies investments.  Create a profile and select which markets you'd like to track!
